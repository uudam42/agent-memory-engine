# Banner Design Specification

## File
`docs/assets/banner.png`

## Dimensions
1600 × 420 px · 72 dpi

## Concept
Dark local-first developer-tool aesthetic.

Visual metaphor: a memory tree / knowledge graph rooted at the centre,
radiating outward as glowing node-edge connections. Left side shows
agent MCP connectivity (dotted arrows); right side shows project
knowledge sources (code, ADR, doc glyphs). The overall tone is calm,
precise, infrastructure-grade — like a circuit diagram rendered in space.

## Typography
- Title: `AGENT MEMORY ENGINE` — heavy weight, 72 px, white, letter-spacing 6 px
- Subtitle line 1: `A LOCAL-FIRST MCP RUNTIME FOR PERSISTENT CODING AGENT MEMORY`
  — 22 px, `#8B949E`
- Subtitle line 2: `Evidence-Backed Memory  ·  Project Knowledge  ·  Agent Context Retrieval`
  — 18 px, `#58A6FF`

## Visual elements
1. Radial node graph, centre-right — 12–15 circles of varying sizes connected
   by thin lines in `#1F6FEB` with 40 % opacity
2. Three larger highlighted nodes labelled: Memory Tree / Knowledge Base / Agent Context
3. Left: dashed arrow entering from outside labelled "MCP stdio"
4. Subtle horizontal scanline texture at 4 % opacity
